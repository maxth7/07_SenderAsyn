#07_SenderAsyn
In that SenderAsynchronous project, executed in C#,

- logging using the Serilog library is used ;
- the library is used CustomCheckBoxLibrary.dll a type    of "Class Library (.NET Framework) dll"
  that implements a custom CheckBox (in repo 08_CustomCheckBox).
